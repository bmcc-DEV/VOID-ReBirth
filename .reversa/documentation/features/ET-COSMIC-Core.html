Requisitos Gerais & Regras de Negócio

Módulo ET-COSMIC-Core (void_core)


Módulo matemático central de segurança e privacidade avançada compilado para WebAssembly.


Visão Geral

O void_core é o núcleo matemático de privacidade e criptografia do ecossistema ET-COSMIC. Desenvolvido em Rust e compilado para WebAssembly (wasm32-unknown-unknown), ele provê primitivas criptográficas pós-quânticas, compromissos homomórficos confidenciais (Pedersen commitments), provas de intervalo em conhecimento zero (Bulletproofs), dispersão de segredos e autenticação física (device binding).


Responsabilidades

  • Geração de Ghost ID: Prover chaves de rede efêmeras e pseudônimas Ed25519 a partir de pools caóticos de entropia.
  • Compromissos Pedersen & Bulletproofs: Ocultar saldos financeiros e validar conservação de valores homomórficos em checkpoints confidenciais.
  • Quantum-Enhanced Layer (QEL): Implementar fragmentação Shamir Secret Sharing e criptografia simétrica ChaCha20-Poly1305.
  • Criptografia PQC: Oferecer algoritmos pós-quânticos ML-KEM-1024 e ML-DSA-87 para comunicação e assinaturas de alta segurança.
  • Auditoria de Licenciamento: Validar payloads de hardware `VOID-00` de forma offline no browser via verificação ML-DSA-87.

Regras de Negócio

  • Curva Ed25519 de Ghost ID 🟢: A chave de identidade deve ser gerada deterministicamente a partir de SHA3-512 sobre o prefixo `void-ghost-id-v8` com entropia de hardware e mapeada na curva base Ed25519.
  • Gerador H Pedersen Nada-na-Manga 🟢: Para garantir independência linear dos geradores da curva elíptica, o ponto $H$ deve ser gerado a partir do hash-to-curve da string constante `hydra-pedersen-H-generator-v8`.
  • Provas de Intervalo a 32 bits 🟢: O valor ocultado no Pedersen deve ser provado via Bulletproofs no intervalo $[0, 2^{32}-1]$ para proteger a CPU em dispositivos móveis.
  • Operação QEL no Corpo Finito 🟢: O polinômio de Shamir opera sobre o Corpo Galois $GF(2^8)$ com polinômio irredutível do AES ($x^8 + x^4 + x^3 + x + 1$).
  • Validação de Handshake offline 🟢: Payloads de ativação comercial devem validar o `device_id` local e janela temporal contra assinatura ML-DSA-87 do vendor.

Design Técnico & Arquitetura

Módulo ET-COSMIC-Core (void_core), Design Técnico


Especificação de design técnico do núcleo matemático WASM.


Interface


O modulo exporta funções WebAssembly compatíveis com binding Javascript/wasm-bindgen.


Símbolos Rust WASM Exportados


| Símbolo | Assinatura (Rust) | Retorno | Observação |

|---------|------------------|---------|------------|

| derive_ghost_id | (entropy: &[u8]) -> GhostId | struct | Retorna chaves Ed25519 e handle. |

| create_pedersen_commitment | (value: u32, blinding: &[u8]) -> Pedersen | struct | Gera o ponto de curva $C = r \cdot G + v \cdot H$. |

| create_balance_proof | (inputs: &[Pedersen], outputs: &[Pedersen]) -> bool | bool | Valida a conservação de valores homomórficos. |

| create_range_proof | (value: u32, blinding: &[u8]) -> Vec | Vec | Retorna bytes da prova Bulletproofs de 32 bits. |

| qel_split_secret | (secret: &[u8], k: u8, n: u8) -> Vec | Vec | Fragmentação Shamir GF(2^8) de segredo. |

| qel_combine_shards | (shards: &[Shard]) -> Vec | Vec | Reconstituição Lagrange de segredo. |

| license_verify_handshake | (payload: &[u8], signature: &[u8], vendor_key: &[u8]) -> bool | bool | Valida payload VOID-00. Entra em pânico se falhar. |


Dependências

  • `curve25519-dalek`: Aritmética de grupo elíptico de alta performance para Ed25519 e Pedersen.
  • `bulletproofs`: Crate de provas de intervalo em conhecimento zero rápidas.
  • `sha3`: Hashing SHA3-512 e SHA3-256 para pools de entropia e Ghost IDs.
  • `chacha20poly1305`: Algoritmo simétrico AEAD para proteção de shards QEL.
  • `wasm-bindgen`: Geração de bindings automáticas para JavaScript do browser.

Estado Interno

O WebAssembly opera em memória linear isolada e controlada. Instâncias de structs como GhostId e Pedersen são expostas como ponteiros opacos gerenciados pelo garbage collector de binding JavaScript, minimizando vazamentos de dados na memória do browser.
