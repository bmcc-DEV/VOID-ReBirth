# Isossupramulação

> **Iso** = paridade com as equações do fenómeno físico.  
> **Supra** = transcendência das limitações materiais via rede ET-COSMIC (mesh, LSC, homotopia, MERA).

Não emula qubits. Executa o fenómeno clássico com fidelidade numérica e escala na malha.

## VOID-600 — Isossupramulated Core

```
┌─────────────────────────────────────────────┐
│         ISOSSUPRAMULATED CORE (VOID-600)     │

└─────────────────────────────────────────────┘
         │ VOID-500 … VOID-506
```

## Filosofia comercial (honesta)

- **Investidor:** otimização até ~10³ variáveis em FPGA/silício — não supremacia universal.
- **Cypherpunk:** canal amarrado à sala (502) — MITM difícil, não impossível com microfone externo.
- **Cientista:** TF ~5–10% vs DFT leve — não substitui ab initio.
- **Anacróclasta:** válvulas virtuais, quartzo, cavidades — sem promessa de qubit.
