## Como Comprar

### Protocol-First ($SOV)
1. Conectar wallet via NWC (`/finance/payment`)
2. Depositar $SOV no ledger
3. Tier ativo baseado no saldo — downgrade automático se saldo insuficiente

### B2B Enterprise (EUR)
1. Contatar MontêLauro Foundation
2. Proposta com `scripts/revenue-calculator.ts`
3. Contrato anual + setup fee (22% do ACV)
4. Build customizado com `node scripts/build-product.mjs <product-id>`
